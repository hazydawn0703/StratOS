# TASK TYPE GUIDE

Defines which Hermes task categories should be routed to StratOS governance in v0.

## Include in v0

### Analysis

- market analysis
- competitor analysis
- scenario/risk analysis

### Planning

- prioritization
- next-step recommendation
- decision memo generation

### Scheduled reports

- daily/weekly summaries
- recurring audits and recommendations

## Exclude in v0

- casual chat and generic Q&A
- pure tool execution
- file movement/format conversion
- tasks with no delayed, reviewable outcome

## Eligibility checklist

A task is eligible only if all are true:

1. Output has durable business impact.
2. Outcome can be observed later (feedback or metric).
3. Task belongs to an allowlisted type.
4. Payload can pass safety/redaction policy.
